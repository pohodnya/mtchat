# Internationalization (i18n)

MTChat supports three languages out of the box with no external i18n dependencies.

## Supported Languages

| Locale Code | Language |
|-------------|----------|
| `ru` | Russian (default) |
| `en` | English |
| `zh` | Chinese (Simplified) |

## Setting the Locale

Pass the locale via the `config` object:

```vue
<script setup lang="ts">
import { MTChat, type MTChatConfig } from '@mtchat/vue'

const config: MTChatConfig = {
  baseUrl: 'https://chat.example.com',
  userId: user.id,
  scopeConfig: { /* ... */ },
  userProfile: { /* ... */ },
  locale: 'en',  // Set UI language
}
</script>

<template>
  <MTChat :config="config" />
</template>
```

## Reactive Language Switching

Change the locale at runtime without remounting the component. Chat state (messages, scroll position, selected dialog) is preserved:

```vue
<script setup lang="ts">
import { ref, computed } from 'vue'
import type { MTChatConfig, Locale } from '@mtchat/vue'

const locale = ref<Locale>('en')

const config = computed<MTChatConfig>(() => ({
  baseUrl: 'https://chat.example.com',
  userId: user.id,
  scopeConfig: user.scopes,
  userProfile: user.profile,
  locale: locale.value,
}))
</script>

<template>
  <select v-model="locale">
    <option value="ru">Russian</option>
    <option value="en">English</option>
    <option value="zh">Chinese</option>
  </select>

  <MTChat :config="config" />
</template>
```

## What Is Translated

All user-facing UI strings are localized:

| Category | Examples |
|----------|---------|
| **Tab labels** | "My Chats", "Available" |
| **Buttons** | "Join", "Send", "Cancel", "Leave Chat" |
| **Status** | "Connected", "Disconnected" |
| **Empty states** | "Select a chat", "No chats available" |
| **Date formatting** | "Today", "Yesterday", locale-aware dates |
| **Message actions** | "Reply", "Edit", "Delete" |
| **System messages** | "created chat", "joined", "left" |
| **File viewer** | Controls, file type labels |
| **Dialog actions** | "Archive", "Pin", "Mute notifications" |
| **Search** | Placeholder text, "No results found" |
| **Chat info** | Participant list, contact labels |
| **Read receipts** | "Read by...", reader names |
| **Error messages** | Connection errors, upload failures |

## Date Formatting

Dates use `Intl.DateTimeFormat` with the selected locale:

- **Today** / **Yesterday** labels use localized text
- Older dates use locale-appropriate formatting (e.g., "Feb 15, 2026" in English, "15 февраля 2026" in Russian)
- Time display follows locale conventions

## Implementation Details

MTChat uses a lightweight i18n system built on Vue's `provide`/`inject`. This approach:

- Requires **no external i18n library** (vue-i18n, etc.)
- Adds minimal bundle size overhead
- Avoids conflicts with your application's i18n setup
- Supports template interpolation for dynamic values (e.g., `"{count} participants"`)

The translation system is internal to the SDK and does not affect your application's locale settings.
