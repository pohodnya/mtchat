# Интернационализация (i18n)

MTChat поддерживает три языка из коробки без внешних зависимостей.

## Поддерживаемые языки

| Код | Язык |
|-----|------|
| `ru` | Русский (по умолчанию) |
| `en` | Английский |
| `zh` | Китайский (упрощённый) |

## Настройка локали

```vue
<MTChat :config="{ ...config, locale: 'ru' }" />
```

## Реактивное переключение

Смена языка не перемонтирует компонент -- состояние чата (сообщения, скролл, выбранный диалог) сохраняется:

```vue
<script setup>
import { ref, computed } from 'vue'

const locale = ref('ru')
const config = computed(() => ({
  ...baseConfig,
  locale: locale.value,
}))
</script>

<template>
  <select v-model="locale">
    <option value="ru">Русский</option>
    <option value="en">English</option>
    <option value="zh">中文</option>
  </select>
  <MTChat :config="config" />
</template>
```

## Что переводится

| Категория | Примеры |
|-----------|---------|
| **Вкладки** | "Мои чаты", "Доступные" |
| **Кнопки** | "Присоединиться", "Отправить", "Отмена", "Покинуть чат" |
| **Статусы** | "Подключён", "Отключён" |
| **Пустые состояния** | "Выберите чат", "Нет доступных чатов" |
| **Даты** | "Сегодня", "Вчера", локализованные даты |
| **Действия** | "Ответить", "Редактировать", "Удалить" |
| **Системные сообщения** | "создал чат", "присоединился", "покинул" |
| **Файлы** | Просмотрщик, типы файлов |
| **Поиск** | Placeholder, "Ничего не найдено" |

## Форматирование дат

Даты используют `Intl.DateTimeFormat` с выбранной локалью:

- **Сегодня** / **Вчера** -- локализованные метки
- Старые даты -- формат по локали ("15 февраля 2026" на русском, "Feb 15, 2026" на английском)

## Реализация

MTChat использует лёгкую i18n-систему на основе Vue `provide`/`inject`:

- Не требует внешних библиотек (vue-i18n и т.д.)
- Минимальный overhead по размеру бандла
- Не конфликтует с i18n-настройками вашего приложения
- Поддерживает интерполяцию шаблонов для динамических значений
