# Полный режим

Полный режим отображает боковую панель со списком диалогов и область чата. Это режим по умолчанию, подходящий для выделенных страниц чата.

## Использование

```vue
<template>
  <div style="height: 600px;">
    <MTChat :config="config" mode="full" theme="light" />
  </div>
</template>
```

!!! important "Высота контейнера"
    Компонент `<MTChat>` заполняет родительский контейнер. Установите высоту на родительском элементе (`height: 600px`, `height: 100vh` или `flex: 1`).

## Вкладки боковой панели

- **Мои чаты** -- диалоги, где пользователь участник (закреплённые вверху, затем по времени последнего сообщения)
- **Доступные** -- диалоги, к которым можно присоединиться по scope-правилам

Ниже активных чатов -- аккордеон **Архив** с архивными диалогами (загружаются лениво).

## Props

| Prop | Тип | По умолчанию | Описание |
|------|-----|--------------|----------|
| `config` | `MTChatConfig` | **обязателен** | Конфигурация SDK |
| `mode` | `'full' \| 'inline'` | `'full'` | Режим отображения |
| `dialogId` | `string` | -- | Предвыбранный диалог |
| `showHeader` | `boolean` | `true` | Показать заголовок чата |
| `showSidebar` | `boolean` | `true` | Показать боковую панель |
| `theme` | `string` | `'light'` | Тема (`'light'` или `'dark'`) |

## События

| Событие | Данные | Описание |
|---------|--------|----------|
| `connected` | -- | WebSocket подключён |
| `disconnected` | -- | WebSocket отключён |
| `error` | `Error` | Произошла ошибка |
| `message-sent` | `Message` | Сообщение отправлено |
| `dialog-selected` | `DialogListItem` | Диалог выбран из списка |
| `dialog-joined` | `string` | Пользователь присоединился |
| `dialog-left` | `string` | Пользователь вышел |

## Слоты

### `sidebar-action`

Кнопка действия в заголовке боковой панели:

```vue
<MTChat :config="config">
  <template #sidebar-action>
    <button @click="createNewChat">+ Новый чат</button>
  </template>
</MTChat>
```

### `header-menu-actions`

Пользовательские пункты в выпадающем меню заголовка:

```vue
<MTChat :config="config">
  <template #header-menu-actions="{ dialog, closeMenu, menuItemClass }">
    <button :class="menuItemClass" @click="openInNewTab(dialog); closeMenu()">
      Открыть в новой вкладке
    </button>
  </template>
</MTChat>
```

## Функции

- **Поиск диалогов** -- Cmd+K (Mac) / Ctrl+K для фокуса, Esc для очистки
- **Контекстное меню** -- правый клик на диалоге: закрепить, архивировать, отключить уведомления
- **Изменяемая боковая панель** -- перетаскивание края для изменения ширины
- **Адаптивность** -- автоматическая адаптация для десктопа, планшета и мобильных устройств
